# Harjutamine
tegemist on versioonihalduse harjutamisega GitHubis.

Mis on GitHub ja kes seda kasutavad

GitHub on veebipõhine platvorm, mis võimaldab arendajatel hoida, hallata ja jagada tarkvaraprojektide lähtekoodi. See põhineb Git-versioonihaldussüsteemil ning toetab koostööd, koodi ülevaatamist ja muudatuste ajalugu.
GitHubi kasutavad tarkvaraarendajad, IT-ettevõtted, õppeasutused ning ka harrastajad, kes soovivad oma projekte arendada ja teistega koostööd teha.
